# Netflix-Movies-and-TV-Shows-Dataset

1) Project Title / Headline:
🎬Netflix Movies & TV Shows Data Analysis Dashboard.

2) Short Description / Purpose:
This project presents an interactive Netflix data analysis dashboard built using Power BI to explore trends and insights from Netflix’s movies and TV shows catalog. The dashboard helps analyze content distribution, release trends, genres, ratings, and country-wise production, enabling better understanding of Netflix’s content strategy and growth over time.

3) Tech Stack:
The dashboard was built using the following tools and technologies:
📊 Power BI – Data visualization and interactive dashboard creation.
🔄 Power Query – Data cleaning and transformation.
🧮 DAX (Data Analysis Expressions) – Creating calculated measures and KPIs.
📁 CSV Dataset – Source dataset containing Netflix movies and TV shows information.

4) Data Source:
The dataset used in this project contains information about Netflix movies and TV shows, including details such as:
🎬 Content Type (Movie or TV Show)
📅 Release Year
🌍 Country of Production
🎭 Genre / Category
⭐ Content Rating
🎥 Director & Cast
📆 Date Added to Netflix
📝 Description
The data is used to analyze content distribution, popularity of genres, and global production trends on Netflix.

5) Features / Highlights:
📊 Content Overview:
Displays total titles, number of movies, and number of TV shows available on Netflix.
📈 Content Growth Analysis:
Shows how Netflix content has increased over the years using time-based visualizations.
🌍 Country-wise Content Distribution:
Identifies top countries producing Netflix content.
🎭 Genre Analysis:
Highlights the most popular genres available on the platform.
⭐ Rating Distribution:
Visualizes content ratings (TV-MA, PG, etc.) to understand audience targeting.
🎛 Interactive Dashboard:
Includes filters and slicers for exploring data by:
Content type, Release year, Country and Rating.
📊 Data Visualization:
Uses KPIs, pie charts, bar charts, and line charts to present insights in an easy-to-understand format.
